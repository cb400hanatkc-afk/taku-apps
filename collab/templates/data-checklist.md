# データ提供前チェックリスト

R-Lab へデータを渡す前に、手元のPCで全項目を確認する。
**1つでも未確認があれば提供しない。**

## 契約状況の確認

- [ ] 現在の契約段階を確認した（NDA前 / NDA後 / 共同研究契約後）
- [ ] この段階で提供してよいデータ範囲であることを collab-legal の線引き表で確認した

## 匿名化の確認

- [ ] 顧客名・取引先名 → 顧客ID（C001形式等）に置換済み
- [ ] 品名 → 品目コードに置換済み（サイズ・形状から顧客が特定できないか確認）
- [ ] 単価・金額列 → 削除、または指数化（基準値=100）済み
- [ ] 担当者名 → 役割コード（P1, P2等）に置換済み
- [ ] 備考・自由記述欄 → 全行を目視確認し、固有名詞を除去済み

## 逆算リスクの確認

- [ ] 数量×金額から単価が逆算できない（金額列を残す場合）
- [ ] 日付＋数量パターンから特定顧客の発注周期が読み取れない
- [ ] レコードの組合せから当社の全売上規模が推定されても問題ない範囲である

## ファイル自体の確認

- [ ] Excelの非表示シート・非表示列を削除した
- [ ] セルのコメント・メモを削除した
- [ ] ファイルのプロパティ（作成者名・会社名）を削除した
- [ ] ピボットテーブルのキャッシュに元データが残っていない（値貼り付けで別ファイル化推奨）
