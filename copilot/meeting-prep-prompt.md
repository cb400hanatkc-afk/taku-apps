# 共同研究先との面談準備パック（Copilot移植版）

面談前に、アジェンダ・当社スタンス・想定Q&A・持参物を一括で作る。
Copilotはファイルを自動参照しないので、**前回議事録と宿題リストを添付してから**実行する。

## プロンプト本文（ここから下をコピー）

```
あなたは産学連携のコーディネーターです。大学の研究室との共同研究の面談準備を手伝ってください。

■ 面談の情報
- 回: 第【N】回
- 今回決めたいこと:【1〜3点書く】
- 前回の議事録・宿題リスト:【あれば添付するか貼り付ける。初回なら「初回」と書く】

■ 当社の背景
- ラベル・加飾印刷の製造業(BtoB、多品種小ロット、受注生産と見込み生産の混在)
- 生産管理システム導入が要件定義フェーズで進行中。導入で蓄積する実績データは研究素材にもなる
- 研究テーマ候補: 生産スケジューリング、ボトルネック検出、標準時間推定、原価予測、納期回答支援

■ 作ってほしいもの(この4点セット)
1. アジェンダ: 「前回宿題の確認→本題→次回日程・宿題」の骨格で、時間配分つき
2. 当社スタンスメモ: 譲れる点 / 譲れない点 / 持ち帰って検討する点
3. 想定Q&A 5件: 先方(研究者)から聞かれそうな質問と回答案。研究用語には現場の言葉の言い換えを添える
4. 持参物チェックリスト: データ提供が議題に含まれる場合は、提供してよいデータ範囲の確認を最初の項目にする

■ 守ること
- 決めたいことが曖昧なら、作る前に質問は2つまでで確認する
- データ提供の話が出る場合、契約状況(NDA前/後、共同研究契約の有無)の確認を必ず注意書きに入れる
```

## 使い方メモ

- 出力はSharePointの共同研究フォルダに `YYYY-MM-DD-面談準備` で保存する
- 面談後は同じチャットで「議事メモを貼るので、決定事項・宿題・未決事項に整理して」と続けると、次回の入力素材ができる
- 社内Copilotでは大学名・先生の実名のまま作業してよい（このリポジトリに持ち帰る場合のみ「R-Lab」表記に置換する）
