# reading/ — Kindle読書リフレクション

Kindleのハイライト・メモのスクリーンショットから読書ノートを作り、
内省とアクションにつなげるための置き場。

## 使い方

1. Kindleでハイライト・メモした画面のスクリーンショットを撮る
   （Kindleアプリの「ノートブック」画面をまとめて撮るのが効率的）
2. Claude Code のセッションにスクショを貼り付けて「読書メモを整理して」と言う
3. スキル `kindle-reflect`（`.claude/skills/kindle-reflect/SKILL.md`）が起動し、
   文字起こし → 捉え方マップ → 内省の対話 → アクション化 → ノート保存まで進む

## 保存先

- **読書ノート本体**: Notionの「著書の要約」ページ配下に、書名をタイトルにしたサブページとして作成
- **アクションリスト**: このリポジトリの `reading/actions.md` に全書籍横断で蓄積

## 構成

| パス | 内容 |
|------|------|
| `actions.md` | 全書籍横断のアクションリスト（蓄積・振り返り用） |
| `notes/` | Notionが使えないセッションでの一時保存先（通常は空） |

## 振り返り

「読書アクションの振り返り」と言うと、`actions.md` の未完了アクションを
1件ずつ確認して状態を更新する。
