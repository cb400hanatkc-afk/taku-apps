# コーチング練習会 自主開催キット

> 目的: アクションプラン②「9月までにコーチング練習オフ会を1回主催する」を、開催できるタイミングが来たら即実行できる状態にしておく。
> 成功基準: 「参加者に大きな気づきを与えたら成功」ではなく、**「場を用意し、問いを投げ、フィードバックから一つ学べたら成功」**。
> 出典: 管理人さん提供のノウハウ記事・投影スライド、および自分の練習会参加メモ5回分（個人名・セッション内容は本書に不使用）。

---

## 1. 開催形態と規約上の注意（最初に確認）

- **1対1のセッション募集はNG**。リベシティでは個人活動のモニター募集が禁止のため、無料モニター行為とみなされるリスクがある。
- **チャット上の学び・交流イベントとして3名以上で開催**すれば規約違反にならない（管理人さん記事より）。
- コーチング部チャットは、規約に反しなければ**各自でイベントを立ち上げてよい**とされている。管理人チームの月1開催と同様の形式で自主開催は公認。
- 守秘義務をルールに含める（セッション内容を場の外に持ち出さない）。

## 2. 募集文テンプレ（コピペ用）

> 【◯月◯日（◯）◯時〜◯時】コーチング練習交流会
>
> ■開催日時
> ◯月◯日（◯）◯:00〜◯:00（2時間）
>
> ■内容
> 3人1組（コーチ役・クライアント役・オブザーバー役）で交代しながらコーチングを実践練習し、お互いにフィードバックし合う会です。
>
> ■練習テーマ（任意で設定）
> 例: 質問・傾聴・承認などの基本スキル practice ／ GROWモデルを意識したセッション ／ セッション冒頭の「ニーズの合意」にフォーカス
>
> ■参加条件
> ・コーチングをやったことがあり、練習したい方
> ・未経験でも、本などで学んで試しにやってみたい方
> ※やり方の細かいレクチャーはしません。各自予習のうえご参加ください。
> ※カメラのオン/オフは自由です（音声のみの参加もOK）。
>
> ■プログラム
> ・オリエンテーション・自己紹介 5分
> ・3人1組ブレイクアウト① 45分
> ・振り返り・感想交流 10分
> ・3人1組ブレイクアウト② 45分
> ・振り返り・感想交流 10分
> ・まとめ・クロージング 5分

## 3. タイムテーブル詳細（120分）

| 時刻 | 内容 | 備考 |
|---|---|---|
| 0:00–0:05 | オリエンテーション・自己紹介 | ルール説明・役割説明・練習宣言の案内 |
| 0:05–0:50 | ブレイクアウト① 45分 | 3人組: 15分サイクル×3（12分セッション+3分FB）または (9+3)×3+バッファ |
| 0:50–1:00 | 全体振り返り・感想交流 | Iメッセージで。良さを伸ばすFB中心 |
| 1:00–1:45 | ブレイクアウト② 45分 | 役割ローテーションを一巡させる |
| 1:45–1:55 | 全体振り返り・感想交流 | |
| 1:55–2:00 | まとめ・クロージング・次回案内 | |

**人数変動時の調整**
- 2人ペアになった組: 1人22.5分（18分セッション＋4.5分FB）×2回
- 4人組になった場合: オブザーバー2名で回す
- 45分の使い方は各組に委ねる（時間配分の裁量を渡す）と運営が軽くなる

## 4. 進行台本（オリエンテーション 5分）

### 4-1. 目的
「実践・フィードバックを通してコーチング力を磨こう」——安心・安全の場で、今の自分の関わり方への気づきを得る。

### 4-2. ルール（スライド投影）
1. **間違っている人はいない**（流派・手法・考え方は色々ある。違いを尊重する）
2. **肯定的な関心を向ける**（その人の中にある答えを引き出す意図で関わる）
3. **答えは相手の中にある**（探求の時間。結論が出なくてもいい）
4. **ここは実験の場**（チャレンジしていい。成功も失敗もない）
5. **守秘義務を守る**（セッションの内容を外に持ち出さない）

### 4-3. コーチングを受けるコツ（クライアント役向け）
- 正解・模範解答を言わなくていい。答えは自分の中にある探求の時間
- 遠慮せず自分の時間として使う
- 黙って考えてもOK（ただし今回の目的は練習なので、そこは考慮を）
- コーチが提案する手法は、まず乗ってみる。抵抗があるときは率直にNGと言ってもいい
- テーマは重すぎない話題を推奨

### 4-4. 3人1組の役割
- **コーチ役**: コーチングをする人（時間管理も担う）
- **クライアント役**: コーチングを受ける人
- **オブザーバー役**: 時間管理・フィードバック。**セッション中は画面オフ＆ミュート**、終わったら画面オン

### 4-5. セッションの流れ
1. 役割を決めてコーチング（ローテーション表）

| ラウンド | A | B | C |
|---|---|---|---|
| 1 | コーチ | クライアント | オブザーバー |
| 2 | オブザーバー | コーチ | クライアント |
| 3 | クライアント | オブザーバー | コーチ |

2. **コーチ役は「今日意識すること」を宣言してから始める**とフィードバックをもらいやすい
3. フィードバックは**オブザーバー → クライアント → コーチの順**で各3分

### 4-6. フィードバックのコツ（スライド投影）
1. **まずリスペクトを忘れない**（色々な手法・考えのコーチングがある）
2. **良かった点を中心に話す**（体験してインパクトがあったところ。良さを伸ばすFBが、その人らしいコーチングを育てる）
3. 気になった所も「**より良くするとしたら**」の観点で伝える
4. **コーチ役からフィードバックを取りに行ってもいい**（試したこと・手法について聞く）
5. **Iメッセージで伝える**（「私は〜と感じた」という情報提供の形）

## 5. スライド準備

管理人さんの共有スライド（Canva）をベースに自分版を作る。必要な構成は7枚:

1. 表紙（タイトル・日付）
2. 目的「実践・フィードバックを通してコーチング力を磨こう」
3. ルール4カ条＋守秘義務
4. コーチングを受けるコツ
5. 3人1組の役割とローテーション表
6. セッションの流れ（宣言→セッション→FB順序・各3分）
7. フィードバックのコツ

## 6. 主催者チェックリスト

### 開催前（2〜3週間前）
- [ ] 開催日時を決める（自分の予定と、部の月1開催・他イベントと被らない日）
- [ ] コーチング部チャットにオフ会イベントを立ち上げ、募集文を投稿
- [ ] スライドを準備（Canva、上記7枚構成）
- [ ] Zoom等の会場URLを確保し、ブレイクアウトルーム機能を確認
- [ ] 最少催行3名。集まらない場合の延期判断日を決めておく（開催3日前）

### 前日〜当日
- [ ] 参加者にリマインド投稿
- [ ] 参加人数から組分け・時間配分を決めておく（3人組×N、端数は2人ペア or 4人組）
- [ ] 自分の「練習宣言」を決めておく（下記7参照）
- [ ] タイマー準備（時間厳守が場の信頼をつくる）

### 終了後
- [ ] 参加のお礼と感想の投稿
- [ ] 自分の振り返りをNotionに記録（もらったフィードバックは「価値の判定」ではなく「より良い場をつくるためのデータ」）
- [ ] **初主催の体験を記事ネタとしてストック**（「どうやって初めて主催したか」はコーチング単体より反応が見込めるテーマ）
- [ ] 次回開催の要否・改善点をメモ

## 7. 自分の練習宣言テーマ候補（過去の参加メモから抽出）

コーチ役をやるラウンドで宣言する「今日意識すること」。これまでの練習で自覚できた課題から選ぶ:

1. **ニーズの合意を最初の5分以内に取る**（「今日のセッションで何が明確になるとよいか」を握ってから進む）
2. **問いを短く、1つに絞る**（問いが長くなる・複数になる癖への対処。1問1答）
3. **「沈黙で受け止める」と「沈黙して放置する」を区別する**（クライアントの独白が続くとき、方向づけの介入を打てるか）
4. **クライアントの言葉をそのまま返す**（自分の構造化・解釈を挟まずミラーリングする）
5. **表情・声のトーン・話すスピードの変化に注意を向ける**（変化の瞬間にフィードバックを返す）

主催回では進行に注意が割かれるので、**宣言は1つに絞る**こと。

## 8. 開催タイミングを逃さないためのトリガー設計

- **リミット**: 9月末までに1回開催（アクションプラン②）
- **逆算**: 募集は開催の2〜3週間前 → **9月開催なら8月中旬〜下旬に告知**
- **管理人さんへの打診（アクションプラン③、8月中旬のオフ会）とは独立して実行する**。管理人になれてもなれなくても、自主オフ会形式で開催できる（規約上も準備上も、待つ理由はない）
- 判断ルール: **8月末時点で開催日が確定していなければ、9月第2週の週末で即決して告知する**（完璧な日程を探さない。このキットがあるので準備リードタイムは不要）
- 部の月1開催と重なりそうなら、あえて近い時期を避けるより「部の開催の2週間後」に置くとリズムが作れる
